این کد بالا، ابتدا تابع shutdown() را تعریف می‌کند که با استفاده از فراخوانی سیستمی (subprocess)، دستوری برای خاموش کردن سیستم را اجرا می‌کند. در این تابع، ابتدا با استفاده از sys.platform، نوع سیستم عامل را تشخیص می‌دهیم و سپس با توجه به نوع سیستم عامل، دستور مناسب برای خاموش کردن سیستم را فراخوانی می‌کنیم.

در بخش اصلی برنامه، ابتدا با استفاده از تابع time.sleep()، 5 ثانیه صبر می‌کنیم تا فرد دست از کامپیوتر بردارد. سپس با استفاده از کتابخانه pyautogui، دکمه ویندوز فشار داده، "cmd" را تایپ کرده و دکمه اینتر را فشار می‌دهیم تا کامند پرومپت باز شود. سپس با توجه به آدرس کامل پوشه فعلی، دستور PyInstaller برای ساخت فایل اجرایی را تایپ می‌کنیم و فایل اجرایی را ساخته و با استفاده از pyautogui، فایل اجرایی را اجرا می‌کنیم. در نهایت، تابع shutdown() را فراخوانی می‌کنیم تا سیستم را خاموش کند.
